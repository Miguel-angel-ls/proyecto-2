Navegadores Web

¿Que es un navegador web?

Un navegador web es un software, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

Tipos de navegadores web

  • Google Chrome
  • Mozilla Firefox
  • Opera
  • Safari
  • Internet Explorer

    		•

    Caracteristicas de los navegadores web

    Todos los navegadores incluyen la mayoría de las caracteristicas: navegación por pestañas, bloqueador de ventanas emergentes, soporte para motores de búsqueda, gestor de descargas.
